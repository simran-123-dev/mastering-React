1. Layout (Dabba kahan rakhein?)
flex: Flexbox on karne ke liye (Sabse important).
flex-col: Items ko upar-neeche stack karne ke liye.
justify-center: Horizontal (left-right) center.
items-center: Vertical (upar-neeche) center.
grid / grid-cols-2: Grid layout ke liye.
gap-4: Items ke beech mein barabar space dene ke liye.
2. Sizing (Kitna bada?)
w-full / w-1/2 / w-64: Width (Percentage ya pixels mein).
h-screen: Poori mobile/laptop screen ki height (100vh).
h-full: Parent element ki poori height.
3. Spacing (Gap kitna ho?)
p-4: Chaaron taraf padding.
px-6: Left-Right padding.
py-2: Top-Bottom padding.
m-auto: Margin auto (Center karne ke liye).
mt-10: Margin Top (upar se thoda gap).
4. Typography (Text kaisa dikhe?)
text-white / text-zinc-400: Font color.
text-xs / text-lg / text-4xl: Font size (chota se bada).
font-bold / font-medium / font-light: Motapa.
text-center / text-left: Text alignment.
italic / uppercase: Text style.
5. Backgrounds & Borders (Sajaawat)
bg-blue-500: Background color (Blue).
bg-transparent: Aar-paar dikhne wala background.
rounded-md / rounded-lg / rounded-full: Corners curve karne ke liye.
border-2: Border ki motai.
border-zinc-100: Border ka color.
6. Special Effects
shadow-lg: Neeche thodi parchaayi (shadow) dene ke liye.
opacity-50: Halki transparency.
hover:bg-red-500: Mouse upar laane par color change.
transition-all: Animation ko smooth banane ke liye.
