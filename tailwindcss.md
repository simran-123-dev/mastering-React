📘 Tailwind CSS Quick Notes
1. 📐 Layout (Dabba kahan rakhein?)
flex → Flexbox enable karne ke liye (sabse important)
flex-col → Items ko vertical (upar-neeche) stack karta hai
justify-center → Horizontal center (left-right)
items-center → Vertical center (upar-neeche)
grid / grid-cols-2 → Grid layout ke liye
gap-4 → Items ke beech equal spacing
2. 📏 Sizing (Kitna bada?)
w-full → Full width
w-1/2 → Half width
w-64 → Fixed width
h-screen → Full screen height (100vh)
h-full → Parent ki full height
3. 📦 Spacing (Gap kitna ho?)
p-4 → All sides padding
px-6 → Left-right padding
py-2 → Top-bottom padding
m-auto → Center align using margin
mt-10 → Top margin (upar se gap)
4. 🔤 Typography (Text kaisa dikhe?)
text-white / text-zinc-400 → Text color
text-xs / text-lg / text-4xl → Font size
font-bold / font-medium / font-light → Font weight
text-center / text-left → Alignment
italic / uppercase → Text style
5. 🎨 Backgrounds & Borders (Sajaawat)
bg-blue-500 → Background color
bg-transparent → Transparent background
rounded-md / rounded-lg / rounded-full → Border radius
border-2 → Border thickness
border-zinc-100 → Border color
6. ✨ Special Effects
shadow-lg → Shadow effect
opacity-50 → Transparency
hover:bg-red-500 → Hover effect (color change)
transition-all → Smooth animation
